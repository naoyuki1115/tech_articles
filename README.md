# Tech Articles

Please check [Qiita ClI](https://github.com/increments/qiita-cli?tab=readme-ov-file)

## Commands

Qiita上の記事のPull
```sh
npx qiita pull
```

記事の新規作成
```sh
npx qiita new 記事のファイルのベース名
```

記事のプレビュー
```sh
npx qiita preview
```

記事の公開
```sh
npx qiita publish 記事のファイルのベース名
# または全公開
npx qiita publish --all
```
